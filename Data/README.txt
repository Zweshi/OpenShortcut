File explanation:
The Shortcut0# files each contain 3 saved text lines and are used by the software to save
and load the users desired shortcut settings. This gives the software the ability to keep
track of the users defined settings between software restarts.

File structure:
Line01: The type of action
Line02: The path address(can be a path to a folder, a file, program or a website)
Line03: The Assigned hotkey key code
